# 機能
- 記録用のファイルの作成
- Excelに出力する出席処理
- 二次元コードでの出席処理
- Excelの色付け
- 出席率の記録
- 出席率のファイル出力　（予定）

# 使用ライブラリ
## pipでのインストールが必要
- PySimpleGUI
- openpyxl
- opencv-python
- cryptography
## python標準ライブラリ
- sys
- os
- datetime
- sqlite3
- time
- json
