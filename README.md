# 機能
- 記録用のファイルの作成
- Excelに出力する出席処理
- 二次元コードでの出席処理
- Excelの色付け
- 出席率グラフの作成　（予定）

# 使用ライブラリ
## pipでのインストールが必要
- PySimpleGUI
- pysqlite
- openpyxl
- opencv-python
## python標準ライブラリ
- sys
- os
- datetime
- sqlite3
- time
- json
