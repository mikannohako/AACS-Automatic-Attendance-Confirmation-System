## このソフトについて

Python初心者が練習などのために作ったソフトです。  
出席の管理などができます。

- 初心者が書いた低クオリティなコード
- ぐちゃぐちゃでカオスなコード

上記のことを許せる方はどうぞ

## 機能

- 記録用のファイルの作成
- Excelに出力する出席処理
- 出席率の記録
- 出席率のファイル出力　（予定）
- 顔認証（いつかやる、、、かも？）
